# Project-2-CS-3502
Project for CS 3502


Open Visual Studio 2022.
Create a new project using the console app template
When the project loads, click on view
Then click on terminal.
Type cd “”Filepath to project” into the terminal
Once you have the correct file path type, “dotnet build”
After it has successfully built type: 
dotnet run --project CpuScheduler/CpuScheduler.csproj
