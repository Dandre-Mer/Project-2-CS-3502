# Project-2-CS-3502
Project for CS 3502


1. Open Visual Studio 2022.
2. Create a new project using the console app template
3. When the project loads, click on view
4. Then click on terminal.
5. Type cd “”Filepath to project” into the terminal
6. Once you have the correct file path type, “dotnet build”
7. After it has successfully built type: 
dotnet run --project CpuScheduler/CpuScheduler.csproj
